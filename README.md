## Comment l'utiliser ?

1. Clone le repo
   ```sh
   git clone https://github.com/ImNeow/FrancorchampsF1.git
   ```
2. Build 
    ```sh
   gcc *c -pthread
    ```
3. Démarrer le prorgramme en choissisant un paramêtre (P1,P2,P3,Q1,Q2,Q3,F)
    ```sh
   ./a.out P1
    ```
